# Get Journal Data

This function retrieves journal data from an existing data file.

## Usage

``` r
get_journal_data(data = "incities")
```

## Arguments

- data:

  A bundled journal source. Use \`incities\` (InCites / JCR) or
  \`scimago\`. The aliases \`incites\`, \`jcr\`, and \`scimagojr\` are
  also accepted.

## Value

A tibble of the journal data.
